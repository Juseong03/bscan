# Abstract

Circular RNAs (circRNAs) are generated by back-splicing, a non-canonical splicing event in which flanking intronic sequences — particularly ALU/SINE repeat elements — facilitate junction formation through reverse-complementary base pairing. Despite growing interest in computational prediction of back-splice sites (BSS), existing deep learning models trained on internal benchmarks fail to generalize across genomic contexts, with performance dropping 39–45% when evaluated on external datasets. We propose **BSCAN** (Back-Splice CircRNA Attention Network), a framework that integrates frozen RNA foundation model (FM) embeddings with a three-branch architecture comprising a 1D-CNN branch, a Watson-Crick stem map branch capturing intron complementarity, and a cross-attention branch encoding upper–lower junction interactions. BSCAN achieves state-of-the-art internal AUC of 0.916–0.917 across four RNA FMs (RNA-FM, RNAErnie, RNABERT, RNA-MSM) using only ~2M trainable parameters, outperforming all published baselines including CircCNN (0.898) and CircDC (0.862). Critically, frozen FM embeddings reduce the internal-to-external AUC drop from 39–45% (CNN-only baselines) to ~8%, and an architectural control (BSCAN-onehot, identical architecture without FM) shows a 37.6% drop, confirming that this generalization advantage is attributable specifically to the FM representation rather than architecture. To mechanistically assess what models have learned about intronic pairing, we introduce a three-tier hard-negative probing framework. CNN-based models discriminate ALU-rich back-splice introns from ALU-poor linear-splice introns (Tier 2 AUC up to 0.727), but no standard-trained model can distinguish individual back-splice introns (Tier 3 AUC 0.49–0.56). Hard-negative augmented training demonstrates that intron-pairing specificity is learnable from sequence data alone (one-hot models: Tier 3 AUC 0.84), yet in the tested frozen-embedding setting, FM models remain near chance even under explicit hard-negative supervision (FM-hnaug Tier 3: 0.509), suggesting that intron-specific signals are not readily recoverable by the downstream classifier in this configuration. Combination with ViennaRNA thermodynamic duplex energy provides a further orthogonal gain (+0.002–0.003 AUC; +0.011–0.017 AUPRC). Integrating cross-dataset generalization, motif safety, thermodynamic signal, and hard-negative robustness, BSCAN-RNA-FM+duplex achieves the highest composite biological profile score. These results establish BSCAN as the recommended framework for cross-context back-splice site prediction, while revealing a fundamental tension in FM-based RNA models between exon-bias resistance and intronic signal accessibility.
